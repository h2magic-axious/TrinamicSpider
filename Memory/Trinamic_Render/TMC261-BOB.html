TMC261-BOB

简介

TMC261转接板是带有专用TMC261-PA芯片的开发板。它的设计目的是让用户有机会快速原型他们的应用程序-使它有可能立即检查如何与TMC芯片的电机执行，同时开发应用程序的软件™酷步™撒布车™

主要参数

    Motor Supply 9…48V
    Phase Current, RMS 1.4A
    Microsteps 1…256
    SPI Yes
    Step/Dir Yes
    StallGuard2™ Yes
    CoolStep™ Yes
    SpreadCycle™ Yes
    产品状态 试产
