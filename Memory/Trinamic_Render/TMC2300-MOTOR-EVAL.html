TMC2300-MOTOR-EVAL

简介

TMC2300-MOTOR-EVAL允许对TMC2300-LA步进电机驱动器进行评估，该驱动器与一个小型内置永磁步进电机相结合，仅由一个锂离子驱动牢房。特别的功能Microplyer™酷步™隐身2™挡泥板4™

主要参数

    Motor Supply 3.3…4.75V
    Phase Current, RMS 1.2A
    Microsteps 1…256
    UART Yes
    USB Yes
    TMCL™ Yes
    MicroPlyer™ Yes
    CoolStep™ Yes
    StealthChop2™ Yes
    StallGuard4™ Yes
    产品状态 量产
