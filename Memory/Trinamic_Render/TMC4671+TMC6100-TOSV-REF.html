TMC4671+TMC6100-TOSV-REF

简介

TMC4671+TMC6100-TOSV-REF是一个针对呼吸机/呼吸器系统的开源参考设计。它是一个BDLC伺服驱动器，电压从+12V到+36V，电机RMS相电流高达6A。它提供了一个广泛的连接，其SBC连接器在Raspberry Pi风格，附加板标题为标准或自定义压力传感器附加板，和通用I/O。通信通过标准TTL UART或RS485（当使用标准附加板TOSV-SENSOR-REF时）。Trinamic开源呼吸机项目

主要参数

    Motor Supply 12…36V
    Phase Current, RMS 6.0A
    Microsteps
    Analog Hall Yes
    RS485 Yes
    UART Yes
    TMCL™ Yes
    产品状态 量产
