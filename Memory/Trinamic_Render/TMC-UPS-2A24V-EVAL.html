TMC-UPS-2A24V-EVAL

简介

TMC-UPS-2A24V-（A）-EVAL/TMC-UPS-10A70V-（A）-EVAL是一种通用电源舞台。它驱动高达2A/10A的电流，并能承受高达24V/70V的电源电压。它使用半桥栅极驱动器和分立MOSFET以及用于电源和电机连接的标准连接器。电流测量由内联分路实现放大器功率级TMC-UPS-2A24V-A-EVAL和TMC-UPS-10A70V-A-EVAL（并联放大器的5V电源）优化用于TMC4671-评估功率级TMC-UPS-2A24V-EVAL和TMC-UPS-10A70V-EVAL（并联放大器的3.3V电源）优化用于TMC8670-EVAL。

主要参数

    Motor Supply 0…24V
    Phase Current, RMS 2.0A
    PWM Yes
    SPI Yes
    Width 85mm
    Length 55mm
    Height 15mm
    产品状态 量产
