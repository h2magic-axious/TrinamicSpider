Landungsbrücke

简介

Landungsbrücke是您的台式电脑和Trinamic芯片解决方案之间的连接。只需连接所需的板组合，并开始发现Trinamic的独特特点Landungsbrücke的核心是飞思卡尔MK20 cortex M4处理器。单一固件设计涵盖Trinamic评估系统的所有评估板。用户可以下载最新版本，修改固件或创建自己的，并通过USB轻松更新。Landungsbrücke的代码是开源的请下载请注意，所有官方固件版本都可以在下面下载。但是，我们也在GitHub页面上发布了夜间构建版本，您可以在这里找到这里：Trinamic GitHub
