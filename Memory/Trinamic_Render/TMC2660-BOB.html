TMC2660-BOB

简介

TMC2660转接板是带有专用TMC2660-PA芯片的开发板。它的设计目的是让用户有机会快速原型他们的应用程序-使它有可能立即检查如何与TMC芯片的电机执行，同时开发应用程序的软件™酷步™撒布车™

主要参数

    Motor Supply 9…30V
    Phase Current, RMS 2.0A
    Microsteps 1…256
    SPI Yes
    Step/Dir Yes
    StallGuard2™ Yes
    CoolStep™ Yes
    SpreadCycle™ Yes
    产品状态 量产
