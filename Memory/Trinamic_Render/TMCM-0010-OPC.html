TMCM-0010-OPC

简介

TMCM-0010-OPC是一个简单的过电压保护模块，有助于将电压轨限制在可配置的最大水平。通过切换连接到的电阻负载（制动电阻器）来限制电压模块可使用Trinamic的TMCL-IDE通过USB配置模块。

主要参数

    Axes 1
    Motor Supply 9…48V
    Phase Current, RMS
    Microsteps
    USB Yes
    Width 45mm
    Length 39mm
    Height 9mm
    产品状态 量产
