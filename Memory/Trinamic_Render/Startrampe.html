Startrampe

简介

Startramp是您的台式电脑和Trinamic芯片解决方案之间的连接。只需连接所需的板组合，并开始发现Trinamic的独特特点Startramp的核心是STM F205 cortex M3处理器。单一固件设计涵盖Trinamic评估系统的所有评估板。用户可以下载最新版本，修改固件或创建自己的，并通过USB轻松更新。代码是开源的请下载访问Trinamic的GitHub页面获取最新信息固件：Trinamic GitHub

主要参数

    Motor Supply 7…70V
    Phase Current, RMS
    Microsteps
    PWM Yes
    RS232 Yes
    SPI Yes
    Step/Dir Yes
    UART Yes
    USB Yes
    产品状态 量产
