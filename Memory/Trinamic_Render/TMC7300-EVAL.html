TMC7300-EVAL

简介

TMC7300-EVAL允许您浏览TMC7300的所有功能-洛杉矶TMC7300-EVAL-KIT由一块MCU板、一块Eselsbrücke桥板和一块TMC7300-EVAL驱动板组成。它允许在短短几分钟内快速简单地启动，使您的电机转动，同时仍然提供对所有寄存器的完全访问和完整的功能和功能诊断。用于诊断时，可以通过桥板上的探头访问MCU和TMC7300之间的所有信号。TMC7300-EVAL可与或不与Landungsbrücke MCU板一起使用。全面的示例代码可以在Trinamics技术访问包中找到好的，好的你需要开始是一个电源和Trinamic的免费和易于使用的TMCL-IDE。此图形用户界面允许快速设置，并支持在直接模式下使用命令，监视图形中可视化的实时行为，以及记录和存储数据。所有设置都可以简单地导出为您自己的固件项目。更多关于TMCL IDE请在使用TMC2300-EVAL时使用Landungsbrücke固件的最新夜间版本，您可以找到这里：Trinamic GitHub
