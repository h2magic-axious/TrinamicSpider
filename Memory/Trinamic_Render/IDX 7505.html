IDX 7505

简介

IDX7505是一个小型和坚固耐用的步进/方向步进电机驱动系统，电源电压高达75V，电机线圈电流高达5A RMS。使用带有ASCII协议的RS-485接口（例如，使用超级终端），可以轻松更改IDX 7505的参数，如电机电流和μ阶跃分辨率。模块的固件也可以通过串行接口进行更新。所有设置都存储在一个内部EEPROM中，因此应用程序中不需要总线系统。该模块是完全保护的，并带有一个独立的参考搜索功能，使用集成的参考开关输入或StallGuard™.所有你需要开始的是一个电源和Trinamic的免费和易于使用的TMCL-IDE。此图形用户界面允许快速设置，并支持在直接模式下使用命令，监视图形中可视化的实时行为，以及日志和存储数据。更多关于TMCL IDE特殊功能StallGuard™
